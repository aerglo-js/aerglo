# Usage

<!-- usage -->

```sh-session
$ npm install -g @aerglo-js/aerglo
$ aerglo COMMAND
running command...
$ aerglo (-v|--version|version)
@aerglo-js/aerglo/0.0.2 darwin-x64 node-v10.15.3
$ aerglo --help [COMMAND]
USAGE
  $ aerglo COMMAND
...
```
